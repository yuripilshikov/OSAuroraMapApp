# Map App

Приложение, отображающее карту.

## Функционал

- Размещение точек при клике мышью;
- Перемещение карты drag and drop;
- Изменение масштаба с помощью ползунка;
- Вкл/выкл отладочной информации (чекбокс с изображением шестерни)
- Вкл/выкл привязки к текущим координатам (чекбокс с изображением стрелки)
